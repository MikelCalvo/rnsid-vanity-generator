## ğŸ“– Overview

rnid-vanity-generator is a bash script that searches for vanity identities generated by **rnid** with specific patterns.  
It uses parallel worker processes, shows progress, estimates runtime, and terminates when a match is found.  
Upon finding a match, it copies the winning identity file into `result_id` and deletes all temporary files.
